How to get a Drive sharing link:

  1. Open Google Drive and find your file
  2. Right-click the file → Share → Copy link
  3. Make sure access is set to "Anyone with the link"
  4. Paste the link below

Paste multiple links separated by commas or new lines. Each must be an individual file link — folder links are not supported.

Embed a YouTube or Vimeo video:

  1. Go to YouTube or Vimeo and find your video
  2. Copy the URL from your browser's address bar
  3. Paste it below — works with standard watch URLs

The video stays hosted on YouTube/Vimeo. The portal embeds it for viewing.
